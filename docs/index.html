Package: tugas2

This package contains classes for managing different types of members in an educational system.

Classes:

  • Member - Abstract base class for all member types
  • Mahasiswa - Student member class
  • Dosen - Faculty member class
  • Karyawan - Staff member class
  • Admin - Administrator class
  • Main - Main application class
