Class Description

Kelas abstrak Member adalah kontrak dasar untuk semua jenis anggota (Dosen, Karyawan, Mahasiswa). Menyediakan atribut umum seperti nama, alamat, dan email, serta metode abstrak untuk input data dan notifikasi.
